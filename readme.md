## Installation

**Required modules**

```
discord.py
asyncio
tinydb
bs4
```

Install the modules by running the following commands in terminal:

```
pip install discord
pip install asyncio
pip install tinydb
pip install bs4
```

## Execution

**Running the application**

From the *SECRETARY_BACON* directory, execute the file `run.py`
